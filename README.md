# gathos-skill

Pre-built agent skills powered by the [Gathos](https://gathos.com) APIs. One install command, production-quality creative output. No glue code. No prompt engineering.

Works with Claude Code, Cursor, Windsurf, Gemini CLI, Aider, and any shell-capable AI agent.

---

## Install any skill

```bash
curl -sL https://gathos.com/skills/<skill-name>.md
```

Paste the output into your agent's context, or save it to your agent's skills directory.

**Requires:** A Gathos API key set as `GATHOS_API_KEY`. Get one at [gathos.com/dashboard](https://gathos.com/dashboard).
